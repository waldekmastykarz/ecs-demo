# Session Evaluations Web App

A vanilla JavaScript web application that displays session evaluations from event attendees. The app shows feedback with classification labels.

## Features

- Display session evaluations with feedback text
- Categorize feedback with visual labels (For Organizers, For Attendees, Useless)
- Clean, modern UI
- Responsive design that works on mobile and desktop
- Dark mode support based on system preferences

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser

## Project Structure

```
├── index.html         # Main HTML file
├── css/
│   └── styles.css     # Styles with modern design
├── js/
│   └── app.js         # Application logic and mock data
```
